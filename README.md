# Customer Registration App

A .NET Blazor application for customer registration management.

## Prerequisites

- .NET SDK (Version 8.0 or later recommended)

## Getting Started

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd CustomerRegApp
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd CustomerRegApp
    ```

3.  **Run the application:**
    ```bash
    dotnet run
    ```

4.  **Access the app:**
    Open your browser and navigate to `http://localhost:5128` (or the port indicated in the terminal).

## Project Structure

- `CustomerRegApp/`: Main project source code.
- `CustomerRegApp.sln`: Solution file.

## Technologies

- ASP.NET Core Blazor
- Entity Framework Core
- SQLite (Default database)
